## 🎨 Cómo Usar

### Opción 1: Copiar URL Directamente
1. Copia la URL de arriba
2. Al crear/editar producto, pega en el campo "URL de Imagen"
3. La imagen se mostrará automáticamente

### Opción 2: Buscar en Unsplash
1. Ve a [unsplash.com](https://unsplash.com)
2. Busca el tipo de producto (ej: "chocolate cake")
3. Abre la imagen que te guste
4. Click derecho en la imagen → "Copiar dirección de imagen"
5. Pega la URL en el campo de imagen
6. Agrega parámetros: `?w=400&h=300&fit=crop` al final

### Opción 3: Unsplash Source API
Formato: `https://source.unsplash.com/400x300/?{keyword}`

Ejemplos:
```
Pastel aleatorio:
https://source.unsplash.com/400x300/?cake

Galletas aleatorias:
https://source.unsplash.com/400x300/?cookies

Donas aleatorias:
https://source.unsplash.com/400x300/?donuts

Pan aleatorio:
https://source.unsplash.com/400x300/?bread
```

## 📐 Dimensiones Recomendadas

- **Tarjetas de producto (POS):** 400x300px
- **Miniaturas (tabla):** 60x60px
- **Vista previa (modal):** 400x300px

## 💡 Tips

1. **Usa `fit=crop`** para que la imagen se ajuste perfectamente
2. **Mantén dimensiones consistentes** para mejor apariencia
3. **Prueba la URL** antes de guardar el producto
4. **Usa imágenes de alta calidad** de Unsplash
5. **Evita imágenes muy pesadas** para mejor rendimiento
