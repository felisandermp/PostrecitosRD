## 🔧 Cómo Cambiar una Imagen

Si una imagen no carga:

1. **Identifica el producto** con problema
2. **Busca en esta lista** las alternativas
3. **Edita el producto** en la aplicación
4. **Copia y pega** una URL alternativa
5. **Guarda** y verifica que cargue

## 💡 Tips para Imágenes que No Cargan

### Problema: Imagen en negro o no carga
**Soluciones:**
- Usa una URL alternativa de esta lista
- Verifica que la URL tenga `?w=400&h=300&fit=crop`
- Prueba la URL en el navegador directamente
- Usa el formato correcto: `https://images.unsplash.com/photo-...`

### Problema: Imagen muy lenta
**Soluciones:**
- Reduce el tamaño: `?w=300&h=225&fit=crop`
- Usa formato webp: `?w=400&h=300&fit=crop&fm=webp`
- Agrega calidad: `?w=400&h=300&fit=crop&q=80`

### Problema: Imagen distorsionada
**Soluciones:**
- Usa `fit=crop` en lugar de `fit=scale`
- Mantén proporciones 4:3 (400x300)
- Verifica que la imagen original sea de buena calidad

## 🔍 Verificar si una URL Funciona

Antes de usar una URL en la aplicación:

1. **Copia la URL**
2. **Pégala en el navegador**
3. **Verifica que la imagen cargue**
4. **Si carga correctamente, úsala en la app**

## 📝 Reportar Problemas

Si encuentras una imagen que no funciona:

1. Anota el nombre del producto
2. Anota la URL que falla
3. Prueba con una alternativa de esta lista
4. Actualiza el producto en la aplicación
